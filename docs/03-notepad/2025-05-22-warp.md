## 1.1.1.1 科学安全上网

1. WARP官网地址: 
`https://one.one.one.one/zh-Hans/`

2. MAC打开终端/ Win 电脑打开CMD窗口，然后输入下面命令更换MAQSUE协议：
```
warp-cli tunnel protocol set MASQUE
```

3. 设置优先IP端点

```
warp-cli tunnel endpoint set 162.159.199.2:8095 // 命令最后的IP地址和端口可根据需要进行替换：
```
